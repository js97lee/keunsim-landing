# 데일리피스 모바일 랜딩

데일리쿠키처럼 모바일 사용자가 바로 구독권을 탐색하고 구매할 수 있도록 만든 B2C 랜딩 페이지입니다.
PC에서는 512px 폭의 모바일 커머스 페이지로 중앙 정렬되며, 작은 모바일 화면에서는 100% 폭으로 줄어듭니다.

## 구성

- `index.html`: 모바일 구매 전환형 랜딩 페이지
- `styles.css`: 오렌지 브라운 디자인 토큰과 공통 컴포넌트 스타일
- `script.js`: 카테고리, 섹션, 상품 카드를 데이터 기반으로 렌더링
- `assets/course-economic-news.png`: 모든 상품 카드에 공통 적용한 임시 썸네일 이미지

## 주요 흐름

- 오렌지 브라운 히어로와 카카오톡 알림 미리보기
- SVG 픽토그램 카테고리 바로가기
- 좌우 버튼과 스와이프를 지원하는 콘텐츠 카드 캐러셀
- 햄버거 메뉴 드로어와 기능 페이지 미리보기
- 동일 썸네일 이미지가 들어간 영어, 제2외국어, 경제, 지원사업, 교양 상품 카드
- 구독권 구매
- 카카오 가입 및 동의 체크
