# 큰심 모바일 랜딩

데일리쿠키처럼 모바일 사용자가 바로 구독권을 탐색하고 구매할 수 있도록 만든 B2C 랜딩 페이지입니다.
PC에서는 512px 폭의 모바일 커머스 페이지로 중앙 정렬되며, 작은 모바일 화면에서는 100% 폭으로 줄어듭니다.

## 구성

- `index.html`: 모바일 구매 전환형 랜딩 페이지
- `styles.css`: 레드바이올렛 커머스 톤의 모바일 우선 스타일
- `assets/*.svg`: 상품 카드용 썸네일 이미지

## 주요 흐름

- 레드바이올렛 히어로와 카카오톡 알림 미리보기
- SVG 픽토그램 카테고리 바로가기
- 실시간 인기 콘텐츠와 신규 콘텐츠 레일
- 실제 썸네일 이미지가 들어간 영어, 제2외국어, 경제, 교양 상품 카드
- 구독권 구매
- 카카오 가입 및 동의 체크
